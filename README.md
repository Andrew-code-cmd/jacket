
## **ENFANTS** — полнофункциональный интернет-магазин нишевой итальянской одежды. 

##  Возможности

| Функция | Описание |
|---|---|
| Каталог товаров | Фильтрация, сортировка, поиск — без перезагрузки |
| Корзина | Живое обновление через HTMX |
| Административная панель | Управление товарами, заказами, скидками |
